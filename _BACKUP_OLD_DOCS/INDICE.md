## 🎯 INÍCIO RÁPIDO

### Para Visualizar o Site
1. Abra `index.html` no navegador
2. Ou inicie um servidor local: `python3 -m http.server 8000`
3. Acesse: `http://localhost:8000`

### Para Ver os Efeitos
1. Abra `demo-efeitos.html` no navegador
2. Ou acesse: `http://localhost:8000/demo-efeitos.html`

### Para Entender o Projeto
1. Leia `README.md` - Visão geral
2. Veja `APRESENTACAO.md` - Resumo executivo
3. Consulte `GUIA-EFEITOS.md` - Documentação técnica
