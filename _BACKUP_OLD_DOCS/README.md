# 🏗️ Website Bronx Engenharia

Website institucional moderno e alto padrão para a Bronx Engenharia, empresa especializada na construção de galpões industriais, logísticos e comerciais em Santa Catarina.

---

## 📋 Sobre o Projeto

Este website foi desenvolvido para transmitir **força**, **segurança** e **modernidade**, destacando a expertise da Bronx Engenharia na construção de galpões Triple A com modelo chave na mão.

### ✨ Características Principais

- ✅ **Design Moderno e Minimalista** - Visual sofisticado com efeitos premium
- ✅ **Glassmorphism Effects** - Elementos com efeito de vidro fosco
- ✅ **Totalmente Responsivo** - Perfeito em desktop, tablet e mobile
- ✅ **Performance Otimizada** - Carregamento rápido e smooth
- ✅ **Hero Slideshow** - Vídeos e imagens de alto impacto
- ✅ **Microinterações** - Hover effects, magnetic buttons, 3D cards
- ✅ **Animações Premium** - Reveal on scroll, parallax, morphing shapes
- ✅ **SEO Otimizado** - Meta tags e estrutura semântica
- ✅ **Identidade Visual Bronx** - Cores teal (#2ABFAA), tipografia Montserrat
- ✅ **Integração WhatsApp** - Botão flutuante com efeitos

## 🚀 Como Usar

### Opção 1: Abrir Diretamente
1. Abra o arquivo `index.html` no seu navegador
2. Navegue pelas páginas usando o menu
3. Para ver os efeitos, abra `demo-efeitos.html`

### Opção 2: Servidor Local (Recomendado)
Para melhor experiência com vídeos e recursos:

```bash
# Usando Python 3
python3 -m http.server 8000

# Usando PHP
php -S localhost:8000

# Usando Node.js (http-server)
npx http-server -p 8000
```

Depois acesse:
- Site principal: `http://localhost:8000`
- Demo de efeitos: `http://localhost:8000/demo-efeitos.html`

## 🔧 Personalizações Necessárias

### 1. Informações de Contato
Atualize em **TODOS OS ARQUIVOS HTML**:

```html
<!-- Telefone -->
<a href="tel:+5547999999999">(47) 99999-9999</a>

<!-- WhatsApp -->
<a href="https://wa.me/5547999999999">WhatsApp</a>

<!-- E-mail -->
<a href="mailto:contato@bronxengenharia.com.br">contato@bronxengenharia.com.br</a>
```

### 2. Redes Sociais
Atualize os links no footer:

```html
<a href="[URL_DO_LINKEDIN]" target="_blank"><i class="fab fa-linkedin"></i></a>
<a href="[URL_DO_INSTAGRAM]" target="_blank"><i class="fab fa-instagram"></i></a>
<a href="[URL_DO_FACEBOOK]" target="_blank"><i class="fab fa-facebook"></i></a>
```

### 3. Endereço Completo
Se desejar adicionar endereço específico:

```html
<!-- Em contato.html e footer -->
<li><i class="fas fa-map-marker-alt"></i> [Endereço Completo], Santa Catarina, Brasil</li>
```

### 4. Google Maps
Atualize o iframe em `contato.html` com as coordenadas corretas:

```html
<iframe src="[URL_DO_GOOGLE_MAPS]" ...></iframe>
```

### 5. Fotos da Equipe
Substitua os placeholders em `sobre.html`:
- Adicione fotos de Ruan e Lucas nas seções de equipe
- Localizadas em: `02_SOBRE/equipe/`
